1. 此功能可否运用于所有手机？返回顶部

您的手机必须开启 GPRS 和 WAP 功能此外，您的手机还须具备 JAVA 技术。

2. 登入时显示密码错误的提示。怎么办？返回顶部

第一步:您必须确定您使用的网络服务已包括(WAP/GPRS)的上网功能。
第二步:您的手机必须正确设置此功能，此设置通常由网络运行商提供。

4. 什么是 WAP 和 GPRS?返回顶部

WAP (无线应用通讯协定)
手机无线上网技术
GPRS (通用无线技术)
一种在现有GSM网络下的高速数据连接，只根据您所下载的数据量收费，与上网时间长短无关。
